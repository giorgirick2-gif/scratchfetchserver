# scratchfetchserver
Fetches Scratches profile and project pics.

## How to fetch profile pics.

https://scratchfetchserver.onrender.com/fetch/profilepic/user_id <-- THE USER ID IS NEEDED, ITS NOT THE USERNAME, YOU CAN CHECK THE ID HERE: https://api.scratch.mit.edu/users/username <-- RELACE WITH THE ACTUAL USERNAME

## How to fetch project pics.

https://scratchfetchserver.onrender.com/fetch/projectpic/project_id <-- THE PROJECT ID IS AT HERE: https://scratch.mit.edu/projects/bunch_of_numbers <-- THIS IS THE PROJECTS PROJECT ID

Final Result (for me):

https://scratchfetchserver.onrender.com/fetch/projectpic/138432
